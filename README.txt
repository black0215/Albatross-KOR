🐦 Albatross 1.2
Albatross는 Level-5의 게임(요괴워치 시리즈 등)에서 사용하는 ARC0 아카이브 파일을 열고, 수정하고, 저장할 수 있는 전문 데이터 에디터입니다. 이 툴은 단순한 세이브 에디터가 아닌, 게임 내 데이터를 직접 수정하여 자신만의 모드를 만들 수 있는 게임 에디터입니다.

📌 주요 기능
ARC0 아카이브 관리: 파일 열기, 수정 및 무결성을 보존한 저장 지원.

리소스 탐색: 텍스트, 모델, 아이콘, 맵 등 다양한 게임 리소스 확인.

데이터 수정: 요괴의 종족, 공격, 필살기 등 게임 내부 파라미터 편집.

정밀한 저장: 재저장 후에도 원본과 완전히 동일한 구조 보장.

💻 사용 방법 (How to use)
1. 준비 단계
게임 롬을 디크립트(Decrypt)하여 ExtractedRomFs 폴더를 준비합니다.

Albatross.exe를 실행하고 New Project를 클릭합니다.

프로젝트 이름, 대상 게임, 언어를 설정하고 ExtractedRomFs 폴더를 선택합니다.

2. 수정 및 저장
프로젝트를 로드한 후, charaparam 등의 옵션을 클릭하여 요괴 데이터를 수정합니다.

수정이 끝나면 Save를 누릅니다. (저장 중 프로그램이 응답하지 않을 수 있으나 정상적인 과정이니 기다려 주세요.)

3. 모드 적용 (Citra / 3DS 본체)
Citra: 게임 우클릭 -> Open mod location -> romfs 폴더 생성 후 생성된 .fa 파일을 복사합니다.

3DS (Luma3DS): SD카드의 luma/titles/[Title ID]/romfs/ 경로에 수정된 파일을 넣고, 부팅 시 Enable game patching 옵션을 활성화합니다.
